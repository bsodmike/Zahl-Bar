Running in development...

  1. Run bundle install --without=production to ensure you have the required dependencies. This will create a .bundle/config file which will always force the skipping of any gems listed in the production group — this is done to skip the need for mysql as defined in the Gemfile.

  2. Run the migrations, rake db:migrate

  3. Two rake tasks have been created to assist in (a) creating an admin user account and (b) seeing the database with sample data in lieu of the pending CRUD controller/views that are missing in the app.

  4. Once both rake tasks have been executed, perform User.last.update_attribute(:role,"admin")

  5. Download and install wkhtmltopdf and place the binary at /usr/local/bin — this is needed to allow downloading of PDFs from the application.

  6. Fire up the development server via rails s and visit http://localhost:3000
